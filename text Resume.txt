---

### Step 1: Start Zookeeper

Kafka comes bundled with a basic Zookeeper server (used before KRaft mode). Run:

```bash
bin/zookeeper-server-start.sh config/zookeeper.properties
```

This will start Zookeeper on the default port **2181**. Keep this terminal running.

---

### Step 2: Start Kafka Broker

Open another terminal, go into the same Kafka directory, and run:

```bash
bin/kafka-server-start.sh config/server.properties
```

This will start the Kafka broker on port **9092** (default).

---

From here, you can:

* **Create a topic**:

  ```bash
  bin/kafka-topics.sh --create --topic test-topic --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1
  ```

* **List topics**:

  ```bash
  bin/kafka-topics.sh --list --bootstrap-server localhost:9092
  ```

* **Start a producer** (send messages):

  ```bash
  bin/kafka-console-producer.sh --topic test-topic --bootstrap-server localhost:9092
  ```

* **Start a consumer** (receive messages):

  ```bash
  bin/kafka-console-consumer.sh --topic test-topic --from-beginning --bootstrap-server localhost:9092
  ```
